ClassTwin

Classroom Ecosystem Digital Twin

Run what-if experiments on virtual classrooms before applying real policy changes. Simulate a full 14-week semester — class size, feedback timing, workload, temperature, peer learning, and more.

Try Live Dashboard View on GitHub
🎓

Agent-Based Simulation

30+ student agents with individual knowledge, stress, motivation, SES, and grit — plus a lecturer agent with configurable teaching style.

📊

19 Policy Scenarios

Class size, feedback delay, workload, interventions, temperature, class mode (online/hybrid), SES diversity — all configurable.

🔄

Fully Reproducible

Fixed random seed = identical output. All configs serializable to JSON. 40 unit tests passing.

🖥️

Interactive Dashboard

Streamlit app with simulator, executive dashboard, and deep-dive analysis — no coding required.

📚

Grounded in Literature

Parameters from Hattie (2008), Shute (2008), Vygotsky (1978), Duckworth (2007), ASHRAE 55, and more.

🇮🇩

Indonesian Context

Built for Indonesian higher education — relevant to 9M+ students across 4,600+ universities.
